# TIC-TAC-TOE-GAME

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

* Two-player gameplay (X and O)
* Automatic winner detection
* Draw detection
* Reset game functionality
* New game option
* Responsive game board using Flexbox
* Color-coded player moves

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Play

1. Open the game in your browser.
2. Players take turns clicking on empty cells.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Use the **Reset Game** or **New Game** button to start over.

## Project Structure

* `index.html` – Game layout and structure
* `tac.css` – Styling and responsive design
* `toe.js` – Game logic and winner detection
